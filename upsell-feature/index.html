Key Features

All pricing is calculated server-side based on the customer's account type and the data entered into the offer by the admin. The admin can configure the offer with either a dollar amount or percentage discount. Disabled upsells are skipped automatically in the flow.

Back-end Calculation Logic

Entered from Credit Upsell Admin Area

const creditAmount = creditUpsell.creditAmount; //integer, confirm validation const discountType = creditUpsell.discountType; // "dollar" or "percentage" const discountValue = creditUpsell.discountValue; // integer, confirm validation.

Get From Company's Account Type

const companyCreditPrice = company.accountType.pricePerCredit;

Calculate Pricing

const basePrice = creditAmount * companyCreditPrice; const savingsAmount = discountType === "dollar" ? discountValue : basePrice * (discountValue / 100); const finalPrice = basePrice - savingsAmount; const perCreditPrice = (finalPrice / creditAmount).toFixed(2);

Calculate what credits can buy (for display)

const postcards = creditAmount / 1; // Small cards = 1 credit const textMessages = creditAmount / 0.1; // Text = 0.1 credit
